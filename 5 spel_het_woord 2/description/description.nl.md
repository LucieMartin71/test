# spel_het_woord_2

Schrijf een programma dat een woord vraagt aan de gebruiker. Vervolgens spelt het programma dit woord, met alle letters naast elkaar, gescheiden door een spatie. Ook na de laatste letter staat een spatie.






### Voorbeeld 1:

**Input:**
	
	Verdacht

**Output:**
	
	V
	e
	r
	d
	a
	c
	h
	t
