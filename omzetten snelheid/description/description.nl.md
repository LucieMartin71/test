# omzetten snelheid

Maak een functie 'snelheid_in_km_per_uur' die een snelheid in meter per seconde omzet naar km per uur. 



### Voorbeeld 1:

**Input:**
	
	snelheid_in_km_per_uur(89.64921388726107)

**Output:**
	
	322.73716999413983



### Voorbeeld 2:

**Input:**
	
	snelheid_in_km_per_uur(42.176505595930514)

**Output:**
	
	151.83542014534984
