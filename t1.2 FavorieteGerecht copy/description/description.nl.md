# Favoriete gerecht

Vraag aan de gebruiker wat zijn favoriete maaltijd is.

Laat de boodschap verschijnen :”Jouw favoriete gerecht is … . Dat lijkt me wel lekker!”

### Voorbeeld

**Invoer :**

    oma's spaghetti

**Uitvoer :**

    Jouw favoriete gerecht is oma's spaghetti. Dat lijkt me wel lekker!
