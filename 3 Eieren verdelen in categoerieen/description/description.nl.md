# 3 Eieren verdelen in categoerieën

Eieren zijn verkrijgbaar in vier gewichtsklassen:
•	XL (73 gram en meer),
•	L (63 – 72 gram),
•	M (53 – 62 gram)
•	S (beneden 53 gram).
Laat de gebruiker het gewicht van een ei ingeven.
Op het scherm verschijnt vervolgens de categorie.





### Voorbeeld 1:

**Input:**
	
	45

**Output:**
	
	S

### Voorbeeld 2:

**Input:**
	
	73

**Output:**
	
	XL
